# Explorador de Livros - API Open Library

Este projeto é uma aplicação funcional desenvolvida para a disciplina de Linguagem de Programação da Fatec Itu.

## Funcionalidades
- **Busca de Livros:** Consome a API da Open Library para buscar informações literárias.
- **Persistência de Dados:** Utiliza `localStorage` para manter a lista de livros salvos mesmo após atualizar a página.
- **Gerenciamento:** Tabela dinâmica que permite visualizar atributos e excluir itens individualmente.

## Requisitos Técnicos Atendidos
- HTML5, CSS3 e JavaScript Moderno (ES6+).
- Consumo de API externa via `fetch`.
- Documentação de funções com JSDoc.
- Interface responsiva e organizada.

## Como executar
1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
